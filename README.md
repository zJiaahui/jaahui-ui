## 介绍

这是我在学习Vue过程中做的一个UI框架，希望对你有用

## 安装
```bash
npm i --save jiaahui-ui
```
## 快速开始
引入jiaahui-ui
```js
import { Button, ButtonGroup, Icon } from "jiaahui-ui"
```
