    # 关于

    jiaahui-ui

    A Vue.js 2.X UI Toolkit for Web

    jiaahui-ui 是一个基于 Vue 的 UI 组件库，目前包括有 Button、 Input、 Switch、 Icon、 Pager、 Dialog、 Toast、 Layout、 Carousel 等组件。项目持续更新中。

    • [x] Button
    • [x] Input
    • [x] Switch
    • [x] Icon
    • [x] Pager
    • [x] Dialog
    • [x] Toast
    • [ ] Carousel
    • [ ] Preview demo and docs online
    • [ ] Component unit testing

    # 说明

    这是我在学习 Vue 的过程中尝试写的一个简单 UI 框架，我不想重复的造轮子，业内好用且难以替代的轮子有很多，我希望通过这种方式来让自己有一些提升，也希望对你有所有帮助。
